# Worker Entity Textures

Place `worker.png` texture file (64x64 pixels, standard Minecraft player skin format) in this directory.

For now, the game will use the default missing texture (purple/black checkerboard) until you add the texture.

You can use any Minecraft player skin as a placeholder or create custom textures for each worker type later:
- builder.png
- lumberjack.png
- miner.png
- hunter.png
- farmer.png
- fisher.png
- storage.png
- blacksmith.png
